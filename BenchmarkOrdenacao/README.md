# Projeto III - Benchmark de Ordenação 
## Descrição do Projeto
Este projeto realiza uma análise comparativa de desempenho entre os algoritmos **Quick Sort** e **Insertion Sort**, avaliando o tempo de execução em diferentes cenários de entrada.

## Metodologia de Testes
Para garantir a precisão dos resultados e atender aos requisitos do edital:
1.  **Cenários:** Foram testados o **Melhor Caso**, **Caso Médio** e **Pior Caso**.
2.  **Métricas:** Tempo de execução (ms/s) e comparação com a complexidade teórica O(n²).
3.  **Análise Estatística:** O relatório técnico inclui média e desvio padrão das execuções.

## 📂 Estrutura do Repositório
* `/src`: Código-fonte em Java dos algoritmos e das classes de benchmark.
* `/nbproject`: Configurações do projeto para IDEs baseadas em NetBeans.
* `Relatório Técnico`: Documento detalhado com conclusões da análise.
