Descrição dos Projetos
Projeto 1: Implementação de Árvores (BST, AVL, Rubro-Negra) e heurística para o Problema do Caixeiro-Viajante.
Projeto 2: Sistemas de busca (Sequencial, Binária e em Árvore) com análise de desempenho.
Projeto 3: Brenchmark de algoritmos de ordenação (Insertion Sort e Quick Sort).
