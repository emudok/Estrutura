# Projeto III - Benchmark de Ordenação 📊
Este projeto consiste na implementação e análise de desempenho de algoritmos de ordenação, desenvolvido para a disciplina de **Estruturas, Pesquisa e Ordenação de Dados (2026/1)**.

## Descrição do Projeto
O objetivo principal é realizar um benchmark comparativo entre dois algoritmos de ordenação, avaliando o tempo de execução e a eficiência teórica (análise assintótica) em diferentes cenários.

### Algoritmos Implementados:
Algoritmo 1 - Quick Sort
Algoritmo 2 - Insertion Sort

## Metodologia de Testes
Para garantir a precisão dos resultados e atender aos requisitos do edital:
1.  **Cenários:** Foram testados o **Melhor Caso**, **Caso Médio** e **Pior Caso**.
2.  **Métricas:** Tempo de execução (ms/s) e comparação com a complexidade teórica ($O(n \log n)$ ou $O(n^2)$).
3.  **Análise Estatística:** O relatório técnico inclui média e desvio padrão das execuções.
